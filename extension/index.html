Welcome to ComparePro

ComparePro is a web application for product comparison across different online stores.

And now we have an extension that allows you to compare from anywhere!

How To Use

  1. Copy the URLs of the product you want to compare
  2. Click the Compare button to view the comparison
  • Hint: The button will copy the current's page URL directly
